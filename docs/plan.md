# personal-blog 项目计划

## 第一阶段：静态页面

目标：先用 HTML + CSS + JavaScript 做出博客静态页面。

需要完成：
- 首页 index.html
- 文章详情页 post-detail.html
- 后台管理页 admin.html

## 第二阶段：Vue 前端

目标：用 Vue 重构静态页面。

需要完成：
- Vue 项目初始化
- 路由配置
- 首页组件
- 文章详情组件
- 后台管理组件

## 第三阶段：FastAPI 后端

目标：完成文章相关接口。

需要完成：
- 获取文章列表
- 获取文章详情
- 新增文章
- 编辑文章
- 删除文章

## 第四阶段：MySQL 数据库

目标：把文章数据保存到数据库。

## 第五阶段：部署上线

目标：把项目部署到云服务器。
