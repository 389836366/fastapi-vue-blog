# 7月15日理论作业

## 1

对象适合保存什么样的数据？为什么一篇文章适合用对象表示？

回答：对象适合描述“一个东西”的多个属性。因为文章有多种属性属于它自己。

## 2

下面对象中的属性名和属性值分别是什么？

```javascript
const article = {
  title: "学习JavaScript",
  isPublished: true
};
```

回答：属性名是title和isPublished，属性值是学习JavaScript和true。

## 3

如何读取下面对象中的 `date`？

```javascript
const article = {
  title: "我的博客",
  date: "2026-07-15"
};
```

回答：console.log(article.date);

## 4

数组有什么作用？

回答：数组用来存放一组有序的数据，就像一排有编号的格子

## 5

为什么数组的第一项是：

```javascript
articles[0]
```

而不是：

```javascript
articles[1]
```

回答：下标代表的是"距离起点的偏移量"，起点本身偏移为 0，所以第一个元素的下标是 0。

## 6

假设：

```javascript
const articles = [articleOne, articleTwo, articleThree];
```

下面代码分别表示什么？

```javascript
articles[1]
articles[2].title
articles.length
```

回答：第一个表示articles里面的第二个元素，第二个表示articles里面的第三个元素的title的值，第三个表示articles的长度是多少，也就是里面有几个元素。

## 7

下面代码会输出什么？

```javascript
const user = {
  name: "yongyang",
  age: 21
};

console.log(user.name);
```

回答：yongyang

## 8

下面代码会输出多少？

```javascript
const numbers = [10, 20, 30, 40];

console.log(numbers.length);
```

回答：4

## 9

为什么使用“文章对象组成的数组”，比创建大量独立变量更方便？

例如为什么：

```javascript
const articles = [articleOne, articleTwo, articleThree];
```

比下面这样更适合管理文章？

```javascript
const articleOneTitle = "...";
const articleOneSummary = "...";
const articleTwoTitle = "...";
const articleTwoSummary = "...";
```

回答；因为方便且减少代码量
