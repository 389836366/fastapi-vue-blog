# 7月16日理论作业

## 1

循环解决了什么问题？为什么不一直手动写多个 `console.log()`？

回答：循环解决的是重复执行相同代码的问题。当数据多的时候手动写console.log()会累死。

## 2

解释下面代码：

```javascript
for (const article of articles) {
  console.log(article.title);
}
```

其中：

- `articles`是什么？
- `article`是什么？
- 大括号里的代码执行多少次？

回答：articles是之前创建的数组。article是for循环里面的变量。代码执行的次数取决于articles里面的元素有多少。

## 3

如果数组中有5篇文章，下面代码执行几次？

```javascript
for (const article of articles) {
  console.log(article.title);
}
```

回答：5次

## 4

下面代码会依次输出什么？

```javascript
const numbers = [10, 20, 30];

for (const number of numbers) {
  console.log(number);
}
```

回答：10,20,30

### 5

为什么下面代码可以处理不同的文章？

```javascript
for (const article of articles) {
  showArticleInfo(article);
}
```

回答：因为代码用了for循环，把articles数组里面的元素都暂存到article里面，然后利用showArticleInfo函数。

### 6

下面代码的作用是什么？

```javascript
if (article.isPublished) {
  console.log(article.title);
}
```

回答：作用是如果文章是true就输出文章的标题。

### 7

统计数量时，为什么要在循环开始前写：

```javascript
let publishedCount = 0;
```

回答：因为要先创造一个变量啊，不创造新变量怎么让它+1。

### 8

下面两段代码有什么不同？

```javascript
for (const article of articles) {
  console.log(publishedCount);
}
```

```javascript
for (const article of articles) {
  publishedCount = publishedCount + 1;
}

console.log(publishedCount);
```

回答：第一段代码是重复输出publishedCount里面的内容，如果articles这个数组有3个元素，那publishedCount就输出三次。第二段本质是让publishedCount加次数，articles里面有几个元素，publishedCount就加几次。

### 9

如果 `articles` 是空数组：

```javascript
const articles = [];
```



`for...of` 中的代码会执行几次？

回答：一次都不执行。
