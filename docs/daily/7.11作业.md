# 7.11理论题

## 1


为什么访问：http://localhost:5500/frontend/ 时，浏览器会自动显示`index.html`？

回答：因为index的优先级最高

## 2

解释：

```html
<a href="article.html">我的第一篇博客</a>
```

其中 `<a>` 和 `href` 分别负责什么？

回答：<a>是创造一个链接，href是指定这个链接去哪。

## 3

`index.html`、`article.html` 和 `style.css` 位于同一个文件夹，为什么可以写：

```html
<link rel="stylesheet" href="style.css">
```

回答：不就是因为这三个文件都属于同一个文件夹所以才可以这样写吗。

## 4

下面两个选择器有什么区别？

```css
h1
```

```css
.article-detail h1
```

它们各自会影响哪些标题？

回答：第一个代码会影响全部的h1标签。第二个代码只影响article-detail里面的h1.

## 5

你的发布时间是：

```html
<p class="article-meta">发布时间：2026-07-08</p>
```



它会不会同时受到下面两段CSS影响？为什么？

```css
.article-detail p {
  line-height: 1.8;
}
```

```css
.article-meta {
  color: #888;
}
```

回答：不会，因为我这里的p的类名叫article-meta。

## 8

为什么把 `transition` 写在：

```css
.post-title a
```

里面，比只写在：

```css
.post-title a:hover
```

里面更合适？

回答：因为:hover主要是实现鼠标悬停效果的。
