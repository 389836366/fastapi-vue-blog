# 今日理论题

## 1. `console.log()` 有什么作用？输出在哪里查看？

回答：`console.log()` 可以把变量、文字、数字、计算结果等内容输出到浏览器开发者工具的控制台。

## 2. 字符串、数字和布尔值分别是什么？

回答：string：字符串，表示文字，例如 `"博客"`：number：数字，例如 `3`boolean：布尔值，只有 `true` 和 `false`

## 3 . `"3"` 和 `3` 有什么区别？

回答：“3”的类型是字符串，3的类型是数字。

## 4. `const` 和 `let` 有什么区别？

回答：const主要用在值不能变化的场景。let主要用在值要变化的场景。

## 5. 执行下面代码后输出多少？为什么？

```javascript
let count = 3;
count = count + 2;
console.log(count);
```

回答：输出5。因为一开始定义的变量为3，但是后面又给count+2，也就是重新赋值。

## 6. 下面代码中的 `+` 有什么作用？

```javascript
console.log("文章数量：" + articleCount);
```

回答：在“文章数量：”后面加上articleCount里面的内容。

## 7.为什么修改完 `main.js` 后，需要刷新页面才能看到新的输出？

回答：浏览器加载页面时会读取并执行一次 `main.js`。修改文件以后，已经执行过的页面不会自动重新执行新代码，所以需要刷新页面，重新加载并执行最新版JavaScript。
