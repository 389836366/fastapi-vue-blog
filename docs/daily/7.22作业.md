# 7.22作业

1. 数组有什么作用？

   回答：数组用来存放一组有序的数据，就像一排有编号的格子。

2. 对象和数组有什么区别？

   回答：数组存列表（一堆东西的集合），对象存详情（一个东西的多个属性）。

3. 为什么 `articles[0]`是第一篇文章，而不是 `articles[1]`？

   回答：因为数组的下标实际上指的是偏移量，因此第一篇下标为0

4. `articles.length`得到什么？

   回答：数组的长度

5. `articles[1].title`读取的是什么？

   回答：数组articles里面第二个元素的title属性的值

6. 为什么 `articles[3]`得到 `undefined`？

   回答：`articles[3]`读取的是数组中的第四项，但是数组目前只有下标`0`、`1`、`2`，不存在下标为`3`的元素，所以结果是`undefined`

7. `for...of`有什么作用？

   回答：`for...of`用于依次取出数组中的每一个元素，并对每个元素执行一次循环体中的代码。

8. 循环中的 `article`始终代表同一个对象吗？为什么？

   回答：`article`不会始终代表同一个对象。`for...of`每循环一次，就会从`articles`数组中取出当前的一个元素，并暂时交给`article`。因此第一次代表第一篇文章，第二次代表第二篇文章，第三次代表第三篇文章。

9. 为什么修改 `articles[0].isPublished`后，仍然需要重新调用渲染函数？

   回答：`articles[0].isPublished`是JavaScript中的数据，页面状态文字是DOM元素中的内容。普通JavaScript数组、对象和DOM页面之间没有自动绑定，所以修改数组中对象的属性，只会改变JavaScript数据，不会自动更新页面。必须重新调用渲染函数，让它读取最新数据并修改DOM。

10. 完整解释 `renderArticleCount(articles)`如何让页面显示文章数量。

    回答：调用`renderArticleCount(articles)`时，`articles`数组作为实参传给函数，函数中的形参`articleList`代表这个数组。`articleList.length`得到数组中的文章数量`3`，然后程序把它和“共”“篇文章”拼接成“共 3 篇文章”，最后通过`articleCountElement.textContent`把这段文字写进HTML元素，页面便显示文章数量。

11. 如果向数组中增加第四篇文章，`articles.length`会变成多少？

    回答：4
