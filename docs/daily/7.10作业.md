# 第一关：基础概念

## 1

下面的代码有什么作用？

```css
.header-container {
  display: flex;
}
```

它会直接控制 HTML 中的哪些元素？

回答：它的作用是让header-container这里面的元素按照flex的方式呈现。主要控制HTML中的site-brand元素和navbar元素。

## 2

```css
justify-content: space-between;
```

在你的页眉中产生了什么效果？

如果把它删除，页面可能发生什么变化？

1.回答：让我页眉里面的内容两端对齐

2.回答：如果删除，导航栏会和左边的文字紧挨在一起

## 3

下面两个属性分别控制什么？

```css
gap: 30px;
align-items: center;
```

回答：第一行的gap是控制两个盒子之间的距离，第二行的意思是第一交叉轴的对齐方式，这里是居中。

## 4

```css
flex-wrap: wrap;
```

有什么作用？在哪种情况下比较容易观察到它的效果？

回答：他的作用是控制是否换行。在改变浏览器界面大小时就可以看到它的效果。



# 第二关：HTML与CSS关系

## 5

解释下面两段代码为什么能够配合工作：

```html
<a href="#about">关于我</a>
```

```html
<section id="about" class="about">
```



其中：

- `href="#about"` 的作用是什么？
- `id="about"` 的作用是什么？
- `class="about"` 的作用是什么

回答：1.href="#about"的作用是点击可以跳转到id为about的位置上。2. id="about"的作用是给元素设立一个唯一的标志名叫about。3. class="about"的作用是给这个元素取一个类名叫about。

## 6

下面这个选择器会选中哪些元素？

```css
.nav-list a {
  color: white;
}
```

它会不会影响页面里的所有链接？为什么？

回答：不会，因为这个类名选择器的意思是选中nav-list里面的a，而不是整个页面的a。

## 7

下面的代码什么时候生效？

```css
.nav-list a:hover {
  color: #ffd166;
}
```

`:hover` 是什么意思？

回答：当鼠标放到链接上面就会生效。:hover的意思是让鼠标悬停到指定元素上面可以实现指定的效果。

# 第三关：Flex父子关系

观察你自己的 HTML：

```html
<div class="post-list">
  <article class="post-card">...</article>
  <article class="post-card">...</article>
  <article class="post-card">...</article>
</div>
```

## 8

为什么 `display: flex` 应该写在 `.post-list` 上，而不是分别写在三个 `.post-card` 上？

回答：因为flex是控制盒子实现横向排列，因此直接在父容器里面写就行了。（个人理解）

## 9

如果删除：

```css
.post-list {
  display: flex;
}
```

三张文章卡片大概会变成什么排列方式？

回答：上下排列而不是横向排列。
