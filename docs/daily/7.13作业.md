# 7月13日理论作业

## 1

比较运算的结果通常是什么类型？

回答：布尔型

## 2

下面代码分别得到什么结果？

```javascript
5 > 3
5 < 3
5 >= 5
```

回答：true，false，true

## 3

`=` 和 `===` 有什么区别？

回答：=是赋值符号，===是严格等于

## 4

解释下面结构：

```javascript
if (条件) {
  代码A
} else {
  代码B
}
```

什么情况下执行A？什么情况下执行B？

回答：当满足if后面的条件就执行A，不满足就执行B

## 5

下面代码会输出什么？为什么？

```javascript
const studyHours = 1;

if (studyHours >= 2) {
  console.log("达标");
} else {
  console.log("未达标");
}
```

回答：会输出“未达标”这三个字，因为没能满足studyHours >= 2。

## 6

下面代码最终输出多少？

```javascript
const target = 7;
const completed = 4;
const remaining = target - completed;

console.log(remaining);
```

回答：3

## 7

为什么测试 `if...else` 时，应该分别测试一个满足条件的值和一个不满足条件的值？

回答：这样可以看自己的代码有没有错。

## 8

下面代码有什么问题？

```javascript
const isLearning = true;

if (isLearning = true) {
  console.log("正在学习");
}
```

应该如何修改？

回答：直接if （isLearning）就行了。
