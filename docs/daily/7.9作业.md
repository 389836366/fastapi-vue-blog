

# 第一关 

## 1.``HTML`` 和`` CSS ``分别是干嘛的？

HTML：负责网页结构和内容

CSS：负责网页样式和布局

总结：HTML是骨架，CSS是皮囊

## 2.``class``是干嘛的？

解释这两行之间的关系：

```html
<article class="post-card">
```



```css
.post-card {
    background-color: white;
}
```

第一行是HTML，给article这个元素添加一个类名 post-card。

第二行是CSS，为类选择器，用来选中所有class 里包含post-card的元素。

## 3.`margin` 和 `padding` 有什么区别？

margin:外边距，控制盒子和另一个盒子之间的距离

padding：内边距，控制盒子内部的信息和盒子的距离



# 第二关：判断题

## 1.

```css
article {
  background-color: white;
}
```

和

```css
.post-card {
  background-color: white;
}
```

如果我的HTML是：

```html
<article class="post-card">
```

那么这两个选择器都有可能影响它。

这是对还是错?

回答：对，article 和 .post-card 都能影响 <article class="post-card">。如果属性冲突，.post-card通常优先级更高。

## 2

```css
span {
  color: #777;
}
```

会影响页面里所有的 `span`。

这是对还是错？

回答：对，这代码是标签选择器，选择了span这个标签

## 3

```css
.study-plan li {
  margin-bottom: 8px;
}
```

这行代码会影响整个页面所有的 `li`。

这是对还是错？

回答：错，这行代码只影响study-plan里面的 li

## 4

```
.about,
.study-plan,
.contact {
  background-color: white;
}
```

这表示 `.about`、`.study-plan`、`.contact` 三个区域都用同一个背景色。

这是对还是错？

回答：对

# 第三关：读代码题

看这段代码，说出它大概会让页面变成什么样。

```css
.post-title {
  font-size: 22px;
  margin-bottom: 10px;
}

.post-summary {
  color: #555;
  line-height: 1.7;
}

.post-date {
  color: #888;
  font-size: 14px;
}
```

回答：.post-title控制post-title这里面的内容，具体是：让字号变成22px，外边距的底部距离为10px。.post-summary 控制post-summary这里面的内容，具体是：字体颜色为#555，行高为1.7。.post-date 控制post-date里面的内容，具体是：字体颜色为#888，字号为14px



# 第四关：实操题

# 改错题

下面这段 CSS 有重复：

```css
.about {
  background-color: white;
  padding: 24px;
  margin-top: 24px;
  border-radius: 10px;
}

.study-plan {
  background-color: white;
  padding: 24px;
  margin-top: 24px;
  border-radius: 10px;
}

.contact {
  background-color: white;
  padding: 24px;
  margin-top: 24px;
  border-radius: 10px;
}
```

你把它改成更简洁的写法。

回答：

```css
.about, 
.study-plan,
.contact {
  background-color: white;
  padding: 24px;
  margin-top: 24px;
  border-radius: 10px;
}
```
