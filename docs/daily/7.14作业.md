# 7月14日理论作业

## 1

函数有什么作用？为什么不把所有代码都直接写在外面？

回答：函数就是把一段代码打包起来，起个名字，需要的时候叫一声名字就能用。因为写函数可以减少代码量。

## **2**

下面哪一段是定义函数，哪一段是调用函数？

```javascript
function sayHello() {
  console.log("你好");
}
```

```javascript
sayHello();
```

回答：第一段是定义函数，第二段是调用函数。

## 3

只定义函数但不调用，函数里面的代码会不会执行？

回答：不会

## 4

解释参数和实际传入的数据：

```javascript
function showName(name) {
  console.log(name);
}

showName("yongyang");
```

其中 `name` 和 `"yongyang"` 分别是什么？

回答：name是形参，yongyang是要传递进去的数据。

## 5

下面代码输出什么？

```javascript
function add(a, b) {
  return a + b;
}

const result = add(3, 2);
console.log(result);
```

回答：5

## 6

`return` 和 `console.log()` 有什么区别？

回答：return是交出去给程序用，能存到变量里、能继续运算。console.log（）是打印给你看，看完就没了，程序用不了。

## 7

下面代码为什么可以使用不同的数据重复调用？

```javascript
showArticleCount(3);
showArticleCount(5);
showArticleCount(10);
```

回答：因为这是函数。

## 8

下面代码会输出什么？

```javascript
function checkNumber(number) {
  if (number >= 5) {
    return "较大";
  } else {
    return "较小";
  }
}

console.log(checkNumber(3));
```

回答：较小
